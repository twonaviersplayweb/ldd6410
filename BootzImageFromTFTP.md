LDD6410的U－Boot直接支持DM9000，因而可以直接从TFTP下载zImage并通过bootm运行：
运行tftp命令下载在kernel Image：
```
LDD6410 # tftp 0xc0008000 zImage-s3c6410
dm9000 i/o: 0x18000300, id: 0x90000a46 
MAC: 00:40:5c:26:0a:5b
operating at 100M full duplex mode
TFTP from server 192.168.1.111; our IP address is 192.168.1.20
Filename 'zImage-s3c6410'.
Load address: 0xc0008000
Loading: #################################################################
	 #################################################################
	 #################################################################
	 #################################################################
	 #################################################################
	 #################################################################
	 ########################################################
done
Bytes transferred = 2279916 (22c9ec hex)
```
运行bootm引导kernel：
```
LDD6410 # bootm 0xc0008000
Boot with zImage

Starting kernel ...
```
